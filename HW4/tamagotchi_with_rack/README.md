# Tamagothi

Tamagotchi game with Rack

## How it works

* Use git clone to clone repository 

`git clone https://github.com/alinajuly/RubyHW.git`

* Install the required gems (Gemfile)

`gem install bundler`

`bundle install`

* Use rackup

`rackup`

Then go to http://localhost:9292

Login with:
* name: user
* password: password

That's all. Enjoy!
