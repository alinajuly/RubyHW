# Tamagotchi

Tamagotchi game with gem to generate HTML.

## How it works

* Use git clone to clone repository 

`git clone https://github.com/alinajuly/RubyHW`

* Install all of the required gems (Gemfile)

`gem install bundler`

`bundle install`

* Run `game.rb` and play :cat: :dog: :mouse:

### Press for

`1 to play`
`2 to feed`
`3 to give a drink`
`4 to hug`
`5 for a walk`
`6 go to bed`
`7 just wathing`
`8 pet information`
`9 end the game`
